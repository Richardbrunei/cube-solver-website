═══════════════════════════════════════════════════════════════════════════════
                          KEY FEATURES
═══════════════════════════════════════════════════════════════════════════════

POSITIONING:
  • Desktop: Left side, vertically centered with scroll-aware positioning
  • Desktop: Minimum 100px from top (stays below header when scrolled)
  • Mobile: Bottom center, slides up when visible
  • Max height: 95vh (desktop), 40vh (mobile)

CLOSE BUTTON (×):
  • Position: Upper left corner of header
  • Size: 32px × 32px tap target
  • Only visible on mobile (≤768px)
  • Hidden on desktop

BUTTON SIZES:
  • Desktop: 240px × 56px (wide horizontal buttons)
  • Tablet: 240px × 56px (vertical stack)
  • Small mobile: 48px × 48px (compact grid layout)

SPACING:
  • Gap between buttons: 0.4rem (compact)
  • Top padding: 0.75rem
  • Bottom padding: 0.75rem
  • Header margin: 0.5rem

COLORS:
  • Background: Semi-transparent white (98% opacity) with blur
  • Border: Rounded 16px with purple accent
  • Shadow: Elevated card effect
  • Selected: Purple border with subtle glow

HOVER EFFECTS (subtle):
  • Hover: scale(1.03), slight lift (-1px)
  • Active/Press: scale(0.98), pressed-in effect
  • Selected: scale(1.02), purple border glow

INTERACTION:
  • Tap color button → selects color
  • Tap X → closes palette (mobile only)
  • Tap outside → closes palette
  • Scrollable if content exceeds max-height
  • Touch support for mobile sticker selection
